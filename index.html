AI地図

今のAIに「できること」を俯瞰する。テキスト・画像・動画・音声・コード・業務自動化——あなたが思っている以上に、AIはすでに広い領域で動いている。

✍️
テキスト生成・要約・翻訳
文章作成・議事録要約・多言語翻訳をAIが瞬時にこなす。ChatGPT・Claude・Geminiが代表例。
🎨
画像生成・編集
テキストから高品質な画像を秒単位で生成。Midjourney・Stable Diffusion・DALL·Eが牽引。
🎬
動画・音声生成
Soraなど動画生成AIが急拡大。音楽・声のクローンも実用レベルに達している。
💻
コード生成・レビュー
GitHub Copilotは開発者の生産性を55%向上させると報告。ノーコードの壁がなくなりつつある。
🔍
リサーチ・分析
Perplexityが検索を革新。数百ページの資料をAIが数秒で読み解き、要点を抽出する。
⚙️
業務自動化・エージェント
メール返信・スケジュール調整・データ入力をAIが自律実行。「AIエージェント」が実用化段階へ。

02 Introductory

主要AIツールの使い分け

ChatGPT・Claude・Gemini・Perplexity・Midjourney——どれも「AI」だが、得意分野はまったく異なる。「用途×ツール」の最適解を知ると、AIの使い方が一気に変わる。

💬
ChatGPT（OpenAI）
最も普及している汎用AI。文章・コード・分析・画像生成（DALL·E）まで幅広くこなす。無料版あり。
🤖
Claude（Anthropic）
長文処理・コード生成・論理的な回答に強み。最大200,000トークンのコンテキストウィンドウ。
🔎
Perplexity
リアルタイム検索特化型AI。ウェブ情報を引用付きで回答。最新情報の調査に最適。
🌐
Gemini（Google）
GoogleサービスとのシームレスなAI統合。GmailやGドライブと連携して業務効率化を実現。
🎨
Midjourney / Stable Diffusion
画像生成特化型AI。テキストプロンプトから驚異的な品質のビジュアルを秒単位で生成。
📊
無料 vs 有料の違い
有料版では最新モデルへのアクセス・生成速度・1日の利用上限が大幅に向上。月$20前後が相場。

03 Introductory

プロンプトの型

同じAIでも、質問の仕方で出力品質は10倍変わる。「良いプロンプト」には再現性のある型がある。3つの型を知れば、あなたのAI活用は今日から変わる。

🎭
型 1: 役割設定（Role）
「あなたはマーケターとして…」とAIに役割を与える。その専門家の視点・語彙・思考で回答させられる。
📐
型 2: 具体的な条件指定
「500字以内で」「箇条書きで3つ」「初心者向けに」——条件が明確なほど、AIの回答精度は上がる。
💡
型 3: 例示（Few-shot）
「このように→」と出力サンプルを見せる。AIがパターンを学習し、期待通りのフォーマットで出力する。
✅
良いプロンプト vs 悪いプロンプト
「メールを書いて」（悪）→「30代女性向けの新商品案内メールを200字で、柔らかい文体で」（良）
📋
コピペOKテンプレ集
要約・提案・添削・アイデア出し・リサーチの5パターン雛形を提供。すぐに使えるプロンプト集。
🔄
反復改善（Iteration）
一回で完璧を目指さない。「もっと短く」「もっとフォーマルに」と会話を続けるほど精度が上がる。

04 Introductory

AIで変わる日常と仕事

「自分には関係ない」と思っていた人ほど、気づいたときには差がついている。仕事・日常・副業での実際の活用シーンを一気に見渡す。

📧
仕事：メール・文書作成
メール下書き・議事録要約・提案書ドラフト。1時間かかった作業が5分になる体験が急拡大中。
📊
仕事：データ分析・リサーチ
競合調査・市場分析・数値の解釈。AIが仮説を立て、必要な情報を一気に収集・整理する。
🍳
日常：レシピ・旅行・学習
「冷蔵庫の食材でレシピを」「4泊5日の沖縄旅行を計画して」——AIが個人秘書として機能する。
💰
副業：ライティング・デザイン
記事作成・SNS投稿・バナーデザインをAIがサポート。少ない工数で複数案件をこなす副業スタイルが急増。
📱
副業：SNS運用代行
投稿文・ハッシュタグ・コメント返信をAI生成。1人で5アカウントを管理するフリーランサーも出現。
🎓
学習・自己成長
英語学習・資格勉強・スキルアップ。AIが個別チューターとして24時間対応。学習効率が劇的に変わる。

05 Introductory

AIがこれから変える世界

AIの進化には終わりがない。「使う側」に留まり続けるために、これから何が起きるかを知っておく。未来は怖くない——知っている人にとっては、チャンスだ。

🤖
AIエージェントの台頭
「ゴールを与えるだけ」でAIが自律的にタスクを実行。メール返信・調査・予約を自動化する時代が来る。
⚡
進化のスピード
GPT-4登場から2年で性能は10倍以上に。2026年以降もこのペースは続く。「様子見」が最大のリスク。
🔀
「使う側」vs「使われる側」
AIに仕事を奪われるのではなく、AIを使いこなす人に仕事が集まる時代。分岐点は今、ここにある。
🌍
変わる職種・産業
マッキンゼーの試算では2030年までに全業務の30%がAIで自動化可能。変化はすでに始まっている。
📈
学び続ける意義
正解は「全部マスターする」ではなく「触り続ける」。週1時間のAI活用習慣が、1年後の差になる。
🚀
